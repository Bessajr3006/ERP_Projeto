Ajuda

Central de ajuda do sistema. Aqui você encontra orientações rápidas de uso e dicas para atualização.

Atualização / Versão

  • Versão atual do sistema: Carregando...
  • Se não atualizar, pode ser cache do navegador/PWA (Service Worker).
  • No Chrome: DevTools → Application → Service Workers → Unregister e depois “Clear storage”.

Permissões

  • Se algum menu não aparece, o seu perfil pode não ter permissão para o módulo.
  • Administrador/Super Admin pode configurar em “Configuração → Cadastro de Perfis”.

Atalhos de Navegação

  • Use o menu superior para acessar os módulos.
  • No celular, toque no ícone do menu para abrir as seções.

Suporte

Em caso de dúvidas, registre a solicitação com print/tela e informe a versão exibida acima.
